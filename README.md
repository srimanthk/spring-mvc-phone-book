# spring-mvc-phone-book
Phone Book Application - Spring MVC and In Memeory Database (H2) 

Application developed using Spring MVC

Required:
1. Java 1.8
2. Maven 3.5.4

Clone/Checkout the project
1. cd spring-mvc-phone-book
2. mvn spring-boot:run

http://localhost:8080/
